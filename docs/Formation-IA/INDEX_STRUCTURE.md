## 🗂️ Conventions de nommage des fichiers

### Pour les modules de cours

```
NN_Titre_Du_Module.md
│  ├─ NN : Numéro séquentiel (01, 02, 03...)
│  └─ Titre : Description succincte avec underscores
```

**Exemples** :
- `01_Fondamentaux_IA_ML_Ventes.md` ✅
- `02_Algorithmes_Prediction_Classification.md` ✅
- `05_Ethique_Gouvernance_Donnees.md` ✅

### Pour les projets de fin de niveau

```
Projet_[Niveau]_[Departement].md
```

**Exemples** :
- `Projet_Debut_Ventes.md`
- `Projet_Moyen_Support_Client.md`
- `Projet_Expert_Sani_Marc.md`

### Pour les ressources communes

```
[Type]_[Description].md
```

**Exemples** :
- `Glossaire_IA_ML.md`
- `Librairies_Outils_IA.md`
- `Cas_Etude_Best_Practices.md`

## 📖 Contenus prévus par niveau

### 🟢 Niveau DÉBUT (Chaque module : 2-3h)

**5 modules fondamentaux** (structure identique par département) :

1. **Fondamentaux IA/ML** 
   - Concepts de base, terminologie, enjeux
   
2. **Algorithmes de prédiction et classification**
   - Types d'algorithmes utilisables
   
3. **Outils IA disponibles**
   - Logiciels/plateformes adaptées au secteur
   
4. **Atelier pratique**
   - Cas concret du département avec données anonymisées
   
5. **Éthique et gouvernance**
   - Conformité, confidentialité, responsabilité IA

**Totaux** : 10-15h par département

---

### 🟡 Niveau MOYEN (3h par module)

**5-6 modules d'approfondissement + Projet** :

1. **Application spécifique au domaine**
   - Solutions concrètes implémentables
   
2. **Segmentation / Stratification des données**
   - Approche métier pour préparer les données
   
3. **Analyse métier avancée**
   - Optimisation des processus existants
   
4. **Automatisation pratique**
   - Mise en place de solutions semi-automatisées
   
5. **Mesure de l'impact**
   - KPIs, ROI, métriques de succès
   
6. **Adopter l'IA en équipe**
   - Change management, formation collègues

**Totaux** : 16-20h par département

**Projet de fin** : Résoudre un cas d'usage réel du département

---

### 🔴 Niveau EXPERT (2-3h par module)

**5-6 modules stratégiques + Projet ambitieux** :

1. **Modèles ML personnalisés avancés**
   - Fine-tuning, ensemble models, deep learning
   
2. **Architecture données et ML**
   - Pipeline complet production, scalabilité
   
3. **Intégration systémique**
   - Connexion avec ERP (M3, D365), Databricks, Azure
   
4. **Gouvernance et conformité**
   - Risques, audit, security, compliance
   
5. **Stratégie IA d'entreprise**
   - Roadmap, priorisation, business case

6. **Gestion de projets IA**
   - Méthodologies agile pour ML, gestion équipes

**Totaux** : 12-16h par département

**Projet stratégique** : Proposer/implémenter une stratégie IA pour le département

## 🎯 Guide d'utilisation pour chaque rôle

### 👨💼 Responsable de département

**Objectif** : Définir la trajectoire formation de votre équipe

**Étapes** :
1. Lire [Stratégie Formation IA - Vue d'ensemble]/../../../strategie_formation.md
2. Identifier les niveaux pour votre équipe
3. Sélectionner modules prioritaires selon vos enjeux
4. Planifier sur 12 mois (Phase 1 = Début, Phase 2 = Moyen, Phase 3 = Expert)

**Points de départ** :
- Ventes : `Formation-IA/Ventes/Debut/`
- Finances : `Formation-IA/Finances/Debut/`
- Support : `Formation-IA/Support-Client/Debut/`

---

### 👩‍🏫 Formateur/Instructeur

**Objectif** : Animer une formation cohérente et engageante

**Considérations** :
- Chaque module est conçu pour **2-3h** de formation
- Les fichiers incluent : Objectifs, contenu, exemples, exercices, checklist
- Adapter les exemples à Sani Marc (hygiène, désinfection, secteurs)
- Utiliser les cas d'étude de Ressources-Communes pour illustrations

**Exemple de session** :
```
0h00-0h05  → Accueil + agenda
0h05-0h20  → Section 1 : Fondamentaux
0h20-0h40  → Section 2 : Concepts clés + Questions
0h40-1h00  → Section 3 : Cas d'usage concrets
1h00-1h20  → Section 4 : Atelier interactif
1h20-1h30  → Exercices + Checklist post-formation
```

---

### 🎓 Apprenant/Participant

**Objectif** : Progresser selon votre niveau et rôle

**Approche recommandée** :
1. **Vérifier vos prérequis** : Checklist début du module
2. **Lire les objectifs** : Savoir où vous allez
3. **Étudier le contenu** : Prendre notes, poser questions
4. **Faire les exercices** : Appliquer le savoir immédiatement
5. **Compléter la checklist** : Valider l'apprentissage

**Progression typique** :
- Semaine 1 : Module Niveau Début (2-3h) ✅
- Semaine 2-4 : Pratiquer cas d'usage
- Mois 2 : Module Niveau Moyen (3h)
- Mois 3-6 : Appliquer dans le travail réel
- Mois 6+ : Niveau Expert (si applicable)

---

### 🏗️ Architecte IA/CTO

**Objectif** : Piloter la stratégie IA de l'organisation

**Ressources clés** :
- Niveau Expert pour tous les départements
- Ressources-Communes pour vue transversale
- `strategie_formation.md` pour roadmap globale
- Intégration avec M3, D365, Databricks, Azure

**Actions** :
- Valider les projets Expert proposés
- Approuver les données/modèles en production
- Gérer la conformité et governance globale
- Mentorer les champions IA par département

## 📋 Checklist création nouveaux modules

Avant de créer un nouveau module, assurer :

- [ ] **Titre clair** : Commence par NN_ (ex: 01_)
- [ ] **Objectifs** : 3-5 objectifs mesurables en début
- [ ] **Contenu structuré** : Sections numérotées, visuels
- [ ] **Exemples contextuels** : Référence à Sani Marc/secteur
- [ ] **Cas pratiques** : Minimum 2-3 exemples concrets
- [ ] **Exercices** : 2-3 exercices applicables
- [ ] **Ressources** : Liens vers glossaire/tools
- [ ] **Longueur** : 2-3h pour Début, 3h pour Moyen, 2h30 pour Expert
- [ ] **Accessibilité** : Langage clair, non-technique
