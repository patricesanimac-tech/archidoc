## 📋 Template 1 : Business Case IA

**Utilisation** : Avant d'approuver budget IA

```markdown
---
# Business Case : [Nom projet IA]

## Executive Summary (1 page)
- **Problem** : [Quoi il wrong?]
- **Solution** : [Quelle IA?]
- **Benefit** : [$ and timeline]
- **Investment** : [$ cost]
- **ROI** : [% return]

## Current state
- **Process** : [Comment actuellement?]
- **Pain points** : [Limitations?]
- **Cost/time** : [Combien spend maintenant?]

## Proposed solution
- **Technology** : [Quel tool/IA?]
- **Timeline** : [Combien de mois?]
- **Team** : [Qui involve?]
- **Risks** : [Quoi pourrait fail?]

## Financial analysis
- **Investment** :
  - Tool : $X
  - Implementation : $Y
  - Training : $Z
  - Subtotal : $X+Y+Z

- **Benefits** (Year 1) :
  - Revenue increase : $A
  - Cost savings : $B
  - Efficiency gains : $C
  - Subtotal : $A+B+C

- **Net benefit** : $(A+B+C) - $(X+Y+Z)
- **ROI** : Net benefit / Investment * 100%
- **Payback period** : Investment / Monthly benefit

## Success metrics
- [ ] KPI 1 : Target value (measurement method)
- [ ] KPI 2 : Target value
- [ ] KPI 3 : Target value

## Risks & mitigation
| Risk | Probability | Impact | Mitigation |
|---|---|---|---|
| Data quality | Medium | High | Audit + cleanup before |
| Adoption | Medium | Medium | Training + championsmodels |
| Performance | Low | High | Risk mitigated by pilot |

## Recommendation
✅ PROCEED / ⚠️ CONDITIONAL / ❌ HOLD

**Next steps** : [X, Y, Z]
```

## 💬 Template 6 : Escalation Protocol

**Quando Chatbot non capisce** :

```
# Escalation Protocol for support chatbot

## Level 1 : Chatbot handles (target: 70%)
```
Customer: "What are your hours?"
Chatbot: "We're open Mon-Fri 8-5 EST. Something else?"
Status: RESOLVED ✅
```

## Level 2 : Chatbot recognizes limitation (target: 20%)
```
Customer: "My system keeps crashing, I'm so frustrated!!!"
Chatbot detects:
├─ Sentiment: ANGRY
├─ Complexity: HIGH
├─ Intent: TECHNICAL_ISSUE

Chatbot: "I see you're frustrated. Let me connect you 
         to a specialist who can really help. Hold one sec..."

Route to: "Critical Issues" queue
Status: ESCALATED ✅
```

## Level 3 : Human agent handles (target: 10%)
```
Customer is angry/complex/custom
Agent receives:
├─ Context from chatbot
├─ Previous convo transcript
├─ Recommended action
└─ Priority: URGENT/HIGH/NORMAL

Agent: [Personalized resolution]
Status: RESOLVED ✅
```

## Escalation rules
| Condition | Action | Priority |
|---|---|---|
| Sentiment = ANGRY | Escalate immediately | URGENT |
| Chatbot confidence < 40% | Offer to escalate | MEDIUM |
| Customer asks "human" | Escalate in 1 click | NORMAL |
| Issue not in KB (3x) | Auto-escalate + learn | MEDIUM |

## Learning loop
When escalated:
├─ Log reason
├─ Collect resolution
├─ Update KB (prevent next time)
└─ Retrain model monthly
```
